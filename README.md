# Master Key: A Whole-First Learning Method

[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.20649460.svg)](https://doi.org/10.5281/zenodo.20649460)
[![License: CC BY-NC-ND 4.0](https://img.shields.io/badge/License-CC%20BY--NC--ND%204.0-lightgrey.svg)](https://creativecommons.org/licenses/by-nc-nd/4.0/)

A whole-first learning method in which the unit of learning is a complete working
block of full internal complexity, received for use from the first minute.
Understanding is the product of operating the block, not its precondition.
